# TECH-COMPANY-EMPLOYEE-DASHBOARD

##  Project Overview
This Excel-based HR operations dashboard provides a bird's-eye view of a tech company's workforce demographics, productivity baselines, and financial metrics. Built using historical employee records, it synthesizes raw data into interactive visual tables that trace connections between professional experience, compensation, performance reviews, and work environments across major cities.

##  Operational Key Performance Indicators (KPIs)

###  Talent Metrics
* **Total Headcount:** 1,001 active employees.
* **Collective Experience:** 8,088 years of accumulated workforce experience.
* **Remote Adoption:** **515 employees** are currently categorized as remote, showing a strong distributed-team framework.
* **Geographic Hubs:** **Seattle** and **New York** hold the largest concentration of company talent.

###  Compensation & Performance Dynamics
* **Organizational Pay Scale:** Maintains a steady **Average Salary of $91,975**.
* **Top Earning Departments:** **Management** holds the highest average salary benchmark, with **Software** and **Data Science** rounding out the upper tiers.
* **Performance Uniformity:** The company shows balanced production across branches, holding an **Average Performance Score of 8.27/10** with minimal variation between core cities like Austin, Chicago, and San Francisco.

---

##  Built with Microsoft Excel

* **Pivot Table Aggregations:** Powering the behind-the-scenes metrics calculations and categorical filtering.
* **Dual-Axis Combination Charts:** Utilized in the *Performance Vs Experience* chart to show dual variables concurrently via bar-line synchronization.
* **Symmetrical Layout Design:** Formatted with high-contrast, color-coded metric cards at the header for quick executive scanning.
